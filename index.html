What is Popcorn.js?


Popcorn is a javascript library for manipulating open video on the web.

Popcorn.js is a project of Web Made Movies, Mozilla's Open Video Lab. It has been developed by the Center for Development of Open Technology at Seneca College and contributors like you.

Get involved

Please download the source code at mozilla/popcorn-js

You can find our evolving documentation on our wiki. We use Lighthouse for bug tracking. Please file a bug or even better, fix one! We have an active developer community on IRC at irc.mozilla.org/popcorn. Please come say hello!

Download
